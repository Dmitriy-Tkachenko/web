# Канбан доска Vue JS
## Описание проекта
На главной странице отображены 3 колонки. Каждая колонка хранит в себе карточки и определяет статус карточки:

* План
* В работе
* Готово

Между колонками можно перемещать карточки при помощи кнопок и drag-and-drop. Карточки хранят в себе следующую информацию о задаче:

* Заголовок "Задача №n", где n - порядковый номер задачи;
* Текст с описанием;
* Дата и время начала;
* Количество потраченного времени;
* Имя ответственного;
* Кнопки редактирования и удаления/подтверждения.

Вся информация о карточке, кроме даты и времени начала и окончания, а также статуса, задается на этапе ее создания пользователем вручную.
При нажатии в карточке на кнопку редактирования вызывается попап, в котором можно отредактировать карточку.

## Используемые технологии:
Для разработки системы используются соедующие технологии:

* HTML
* CSS
* Vue JS
